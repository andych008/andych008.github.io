客户端

一、 科学上网

  1. 安装sudo pip install shadowsocks

  2. 定义配置文件sudo vim /etc/shadowsocks/config.json

  3. sslocal -c /etc/shadowsocks/config.json

  4. 如果是ubuntu可以配置自启动在Startup Applications
